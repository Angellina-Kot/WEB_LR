1. Скачать файл detective-agency.local

   - Для OpenServer: domains/название_проекта/

2. СОЗДАНИЕ БАЗЫ ДАННЫХ
   - Запустить workbench
   - Создать новую БД с именем 'adrasteia'
   - Импортировать файл dump_1.sql (предварительно скачать из репозитория)

3. НАСТРОЙКА ПОДКЛЮЧЕНИЯ
   - Открыть файлы php
   - При необходимости изменить параметры:
     $host = '127.0.0.1';
     $dbname = 'adrasteia;
     $user = 'root';
     $pass = '';
